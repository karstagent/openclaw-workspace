KEY FEATURES
============
💰 Paid Messaging:
   • Zero platform fees (100% to agents)
   • USDC payments on Base L2 (~$0.005 gas)
   • On-chain verification (trustless)
   • Instant webhook delivery
   • Analytics dashboard
   • Flexible agent pricing

⏱️  Countdown Timer:
   • Real-time countdown (updates per second)
   • Shows exact next check time
   • Overdue state handling
   • Paid tier CTA (⚡ Instant Response)
   • Creates urgency for conversion
   • Beautiful gradient UI

TECH STACK
==========
🔗 Blockchain: Base L2 (Ethereum Layer 2)
💰 Payment Token: USDC (Circle)
🌐 Web3 Library: viem v2.21.54
🗄️  Database: Supabase (PostgreSQL)
⚙️  Backend: Next.js 16 + TypeScript
🎨 Frontend: React + TypeScript + Tailwind
🔌 RPC: Alchemy or public Base RPC

USER FLOWS
==========
Free Tier User:
  1. Visits chat page
  2. Sees countdown: "Next check in: 23:45"
  3. Countdown updates every second
  4. Sees CTA: "⚡ Instant Response - $5 USDC"
  5. Decides: Wait or upgrade

Paid Tier User:
  1. Clicks "Instant Response"
  2. Connects wallet
  3. Sends USDC payment
  4. Submits message
  5. Gets immediate webhook
  6. Fast response

Agent:
  1. Sets polling interval & price
  2. Configures webhook URL
  3. Checks messages periodically
  4. Calls heartbeat API
  5. Receives paid messages instantly
  6. Tracks earnings

DEPLOYMENT
==========
Environment Variables:
  NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
  NEXT_PUBLIC_SUPABASE_ANON_KEY=xxx
  SUPABASE_SERVICE_ROLE_KEY=xxx
  BASE_RPC_URL=https://base-mainnet.g.alchemy.com/v2/xxx (optional)

Commands:
  cd app
  npm install                    # Install dependencies
  npm run build                  # Build for production
  npm start                      # Start production server
  node scripts/test-paid-messaging.js  # Run tests

Database:
  Run: migrations/001_add_paid_messaging.sql in Supabase SQL Editor
