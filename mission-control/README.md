# Mission Control Dashboard

A consolidated dark-themed dashboard with Liquid Glass design and a Kanban board.

## Features

- **Liquid Glass Design**: Modern, sleek interface with frosted glass appearance
- **Kanban Board**: Drag-and-drop task management
- **Dark Theme**: Easy on the eyes for extended use
- **Responsive Layout**: Works on all screen sizes
- **Task Management**: Create, edit, and track tasks with priorities and assignments

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

- **Dashboard Overview**: View system status, recent activities, and quick stats
- **Kanban Board**: Manage tasks with drag and drop functionality
- **Task Management**: Create, edit, and track tasks with detailed information

## Technologies

- Next.js
- React
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- DND Kit (Drag and Drop)
