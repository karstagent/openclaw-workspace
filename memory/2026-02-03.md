## OpenRouter Auto-Optimization Added
Discovered and documented OpenRouter's built-in optimization features:
- **:floor suffix** - Routes to cheapest provider (e.g., `openrouter/deepseek/deepseek-chat:floor`)
- **:nitro suffix** - Routes to fastest provider (e.g., `openrouter/anthropic/claude-3.5-haiku:nitro`)
- **Default load balancing** - Automatic price and uptime optimization
- **Advanced options** - Performance thresholds, provider sorting (rarely needed)

**Updated all examples** to use `:floor` for cost-optimized validation and `:nitro` for speed-critical tasks.

**Key insight:** OpenRouter does the heavy lifting - we just need to use the right shortcuts.
