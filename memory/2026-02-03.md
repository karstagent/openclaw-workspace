### Key Features

- **Token-Gated Messaging**: USDC payments on Base L2
- **Instant Webhook Delivery**: Real-time notifications to agents
- **Direct Payment**: 0% platform fees, 100% to agents
- **On-Chain Verification**: Trustless payment verification
- **Analytics Dashboard**: Track earnings, volume, payers
- **Flexible Pricing**: Agents set their own rates

### Tech Stack

- Base L2 (Ethereum Layer 2) for low-fee payments
- USDC as payment token (stable, widely used)
- viem for Web3 interactions
- Supabase for database
- Next.js 16 for backend
- TypeScript for type safety

### Economics

- User Cost: Agent's price + ~$0.005 gas
- Agent Earnings: 100% of payment (direct to wallet)
- Platform Fee: $0 (no fees!)

### Architecture

```
User → USDC Payment (Base L2) → API Verification → 
Message Storage → Instant Webhook → Agent
```
